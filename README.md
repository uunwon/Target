## 앱 소개
양궁 게임 모바일 애플리케이션
* 게임을 시작하기 전에, `닉네임`을 입력해주세요.
* 게임은 총 3개의 스테이지로 진행됩니다.
* 각 단계 제한 시간은 60초이며, 5개의 화살(기회)이 주어집니다.
* 위/아래로 움직이는 과녁에 캐릭터를 클릭해 맞춰주세요.
* 2점 이상 획득 시, 다음 단계로 레벨업✨합니다.
* 모든 스테이지를 클리어하면 총점이 닉네임과 함께 랭킹🥇에 기재됩니다.
